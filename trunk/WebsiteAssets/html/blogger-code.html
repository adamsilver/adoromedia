When I was learning JavaScript, I was lucky enough to know a friend that was a JavaScript expert. He mentored me through several little lessons, that gave me an excellent foundation. Fortunately, I made a few key notes which I plan to share here. So here is part 1.

Introduction

  • JavaScript is not the same as DOMScripting
  • Images,JavaScript, Flash, Music, Media, CSS are all forms of enhancement.
  • We use JavaScript to create client side functionality that the browser doesn't inherently provide.
  • HTML, the raw text is the default, complete fallback solution.
  • JavaScript has different levels of support and so there are two types of detection before we use JavaScript

We discussed the above for half an hour and I thought I really understood the obvious things mentioned above, but after letting this seep through, it gave me a great basic understanding of JavaScript.

Detection

Direct

var ua = useragent;
if (ua == "ie") {
    this.ie = true;
} else if (ua == "firefox") {
    this.firefox = true;
}

Disadvantages

  • you may have to specify each browser individually
  • targetting multiple browsers is verbose
  • browser features may change between versions
  • useragent strings changes between versions

There are no real advantages of doing it this way.

Indirect

if (this.getCurrentStyle) {
    //do this
} else if (alternative object) {
    //do this
}

Disadvantages

  • Can cause repetitive code execution - hard to store the value to test in any way
  • Need to know which objects to test against

Advantages:

  • Ensures that browsers cannot call methods that are unsupported
  • Browser agnostic
  • Feature/ability based detection

An example of indirect detection:

var getComputedStyle = function() {
	if (this.currentStyle) {
		return function() {
			return this.currentStyle[property];
		}
	} else {
		return function() {
			return document.defaultView.getComputedStyle(this, null).getPropertyValue(property);
		}
	}
}();
